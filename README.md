## Shopy-Chnopy - https://shopy-chnopy.vercel.app/

# Технологии: Next.js, TypeScript, Tailwind CSS, Zustand

# Функционал:
 * Регистрация/авторизация: Использование Firebase
 * Корзина пользователя: Хранение в store и отправка на сервер (Cloud Firestore)
 * Разработка админ-панели для создаеия и редактирования товаров. Данные для админки - (email - Krus@mail.ru, password - 888888)
 * Сортировка, фильтрация, поиск и оценка товаров
 * Использование нейросетей для генерации описания товаров (Copilot)
 * Адаптивный дизайн
